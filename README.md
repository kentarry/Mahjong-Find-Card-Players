## 📖 簡介

湊咖（Còukā）是一個輕量化的麻將揪團媒合平台，讓想打麻將的人可以快速找到牌友或開局招人。透過 Threads/Instagram 帳號登入，發布或搜尋牌局，並直接跳轉至 Threads 私訊聯繫主揪。

**核心理念：** 不自建聊天室、不經手金錢，僅做資訊媒合，將社交互動導流回 Threads 平台。

## 🖥️ Demo

開啟 `index.html` 即可在瀏覽器中體驗完整功能（CDN 載入 React，無需安裝）。

## ✨ 功能

### 🔐 會員系統
- Threads/Instagram OAuth 2.0 登入（Demo 模擬）
- 首次登入強制同意免責聲明與使用條款
- 黑名單機制：封鎖的用戶開的局不再顯示

### 📝 主揪開局
- 地點選擇（縣市/行政區 + 自由填寫場地名）
- 場地類型（公開棋牌社/私人住宅/其他）
- 缺額人數（1-3 人）
- 開打時間（日期 + 時間選擇器）
- 底台金額（快速選擇 + 自由輸入）
- **50+ 條件標籤**，分為 5 大類：
  - 🏠 場地設施：有廁所、有冷氣、有停車位、有電動麻將桌、近捷運站...
  - 🚬 菸酒規定：禁菸、可菸、可電子菸、可飲酒、禁酒...
  - 👤 性別/身份限制：限女性、限男性、不限性別、限年滿18歲...
  - 🀄 牌規：台灣麻將16張、日本麻將、花牌、門清有平胡...
  - 🎯 玩家偏好：新手友善、快手、歡樂場、認真場、教學局...
- 防呆驗證：時間不可早於現在、金額不可為負

### 🔍 牌友找局
- 大廳依時間排序
- 可收合篩選面板：地區、日期區間、時段區間、底注範圍、場地類型
- 標籤快速篩選（交集）
- 狀態標示：🟢招募中、🟡即將開打、🔴已滿局、⚫已結束
- 倒數計時顯示
- 點擊卡片進入詳情頁

### 🤝 媒合流程
- 點擊「我有興趣」→ 跳轉至主揪的 Threads 個人頁
- 由牌友主動私訊主揪聯繫

### 🛡️ 信任與安全
- **完整免責聲明**（7 條條款 + 行為守則）
- 每個牌局詳情頁內建安全提醒 Banner
- 場地類型標示（公開棋牌社 vs 私人住宅）
- 檢舉機制（5 種原因：放鳥、詐騙、騷擾、不實資訊、其他）
- 累計 3 次檢舉自動停權
- 黑名單封鎖/解封管理

### ⚙️ 系統自動化
- 過期牌局自動下架（開打 2 小時後標記為已過期）
- 主揪可手動「標記已滿局」或「刪除」

## 🚀 快速開始

### 直接開啟（最簡單）

雙擊 `index.html` 即可在瀏覽器中執行，無需安裝任何依賴。

### 本地開發（使用 Vite）

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽建置結果
npm run preview
```

### 部署

**GitHub Pages：**
```bash
npm run build
# 將 dist/ 資料夾內容推送到 gh-pages 分支
```

**Vercel / Netlify：**
- Build command: `npm run build`
- Output directory: `dist`
